RGM-79G Federation Gelgoog is a mass production mobile suit developed by Anaheim Electronics for the Earth Federation Forces based on captured Zeon MS-14 Gelgoog data. Part of the G-QUUUUUX project series, it represents the Federation's attempt to adapt Zeon mobile suit technology into their own manufacturing processes.

Development History

Following the One Year War, the Federation acquired several intact Gelgoog units. Under Project G-QUUUUUX, these were studied and rebuilt using Federation technology and manufacturing methods. The resulting RGM-79G combined Zeon performance characteristics with Federation reliability and ease of maintenance.

Design

Combat Characteristics

While retaining the basic performance profile of the original Gelgoog, the RGM-79G features Federation-standard panoramic cockpit displays and linear seat configuration. The mobile suit's reactor output was slightly reduced to improve reliability, though this affected maximum thrust.

Armament

Standard Federation beam rifle replacing the Zeonic model, with compatibility for E-CAP magazines. Twin beam saber design was retained but modified to use Federation power systems. The shield design incorporates both Zeon and Federation design elements.

Systems

Federation-standard sensor package with modified software to handle the Gelgoog's superior mobility. Enhanced cooling systems were added to compensate for the Federation's extended operation requirements.

Operational History

Limited production run as part of the G-QUUUUUX evaluation program. Units were assigned to test squadrons and veteran pilots for comparative analysis against standard GM units. Program data influenced later Federation mobile suit designs.
